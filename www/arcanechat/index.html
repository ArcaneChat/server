Help

Frequently Asked Questions

  • Who is developing ArcaneChat?

    ArcaneChat is developed by Asiel Diaz Benitez (adb), a software developer passionate about bringing privacy, security and freedom to the people. He is also one of the developers of the Delta Chat app.

  • Is ArcaneChat compatible with DeltaChat?

    Yes, ArcaneChat is a Delta Chat client, you can chat with Delta Chat users using ArcaneChat.

  • How long are my messages stored on the arcanechat.me server?

    If you have a single device, your ArcaneChat app will delete messages immediately after downloading them. Messages are unconditionally removed 20 days after arriving on the arcanechat.me server.

  • Account Deletion: How to remove my profile?

    If you uninstall ArcaneChat or remove an arcanechat.me profile from within the ArcaneChat app, then the according account on the server, along with all associated data, is automatically deleted 90 days afterwards.

    If you use multiple devices then you need to remove the according profile from each device in order for all account data to be removed on the server side.

    If you have any further questions or requests regarding account deletion please contact us using your ArcaneChat account.
